 Contact-RxSwift
 ============
 
 ## 사용된 버전
 ### XCode 11.6, Swift 5
 
 ## 사용된 오픈소스 
 ### RxSwift, RxCocoa
* Link : <https://github.com/ReactiveX/RxSwift>

### 연락처를 불러와서, RxCocoa로 테이블 뷰에 뿌려주고, 해당 셀을 클릭했을 시,   전화 연결되는 기능입니다. 
### 우선 Mobile로 등록된 전화가 아닐 경우, 번호가 보이지 않을 수 있습니다.
### 번호가 뿌려지는 형식이 제각각인데, 조만간 다시 수정할 생각입니다.
